Que es Emmet

Emmet es una herramiento que nos ayuda a escribir HTML y CSS de forma mas rapida.

Nodos hijos

Usamos el >.

nav>ul>li

<nav>
    <ul>
        <li></li>
    </ul>
</nav>

Nodos Hermanos

Usamos el +.

h1+p+section

<h1></h1>
<p></p>
<section></section>

Nodos superiores

Usamos el ^.

ul>li>a^li>span

<ul>
    <li><a></a></li>
    <li><span></span></li>
</ul>

nav>ul>li>a^^^main>section

<nav>
    <ul>
        <li><a></a></li>
    </ul>
</nav>
<main>
    <section></section>
</main>

Multiplicadores

Usamos el *.

nav>ul>li*5

<nav>
    <ul>
        <li></li>
        <li></li>
        <li></li>
        <li></li>
        <li></li>
    </ul>
</nav>

Agrupacion

Usamos el ( ).

main>section>(h2>span)+p

<main>
    <section>
        <h2><span></span></h2>
        <p></p>
    </section>
</main>

Atributos id y class

Usamos el # para los id y el . para class.

nav>li.item-menu*3>a#link

<nav>
    <li class="item-menu"><a href="" id="link"></a></li>
    <li class="item-menu"><a href="" id="link"></a></li>
    <li class="item-menu"><a href="" id="link"></a></li>
</nav>

Atributos HTML

Usamos el [ ].

ul>li*3>a[target="_blank"]

<ul>
    <li><a href="" target="_blank"></a></li>
    <li><a href="" target="_blank"></a></li>
    <li><a href="" target="_blank"></a></li>
</ul>

ul>li*3>a[target="_blank" role="link"]

<ul>
    <li><a href="" target="_blank" role="link"></a></li>
    <li><a href="" target="_blank" role="link"></a></li>
    <li><a href="" target="_blank" role="link"></a></li>
</ul>

Elementos enumerados

Usamos el $.

nav>ul>li.item-$*5

<nav>
    <ul>
        <li class="item-1"></li>
        <li class="item-2"></li>
        <li class="item-3"></li>
        <li class="item-4"></li>
        <li class="item-5"></li>
    </ul>
</nav>

Elementos enumerados personalizados

Usamos el @.

nav>ul>li.item-$@100*4

<nav>
    <ul>
        <li class="item-100"></li>
        <li class="item-101"></li>
        <li class="item-102"></li>
        <li class="item-103"></li>
    </ul>
</nav>

Contenido HTML

Usamos el { }.

h1{Estamos aprendiendo Emmet}

<h1>Estamos aprendiendo Emmet</h1>

nav>ul>li*4{Seccion $}

<nav>
    <ul>
        <li>Seccion 1</li>
        <li>Seccion 2</li>
        <li>Seccion 3</li>
        <li>Seccion 4</li>
    </ul>
</nav>
